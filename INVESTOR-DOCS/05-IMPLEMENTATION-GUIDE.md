### Key Formulas

**User Growth:**
```
Month N Users = Month N-1 Users Ã— (1 + Growth Rate)
```

**Revenue:**
```
MRR = Paying Subscribers Ã— Average ARPU
ARR = MRR Ã— 12
```

**CAC:**
```
CAC = Marketing Spend / New Customers Acquired
```

**LTV:**
```
LTV = Average ARPU Ã— Average Retention (months)
```

**Gross Margin:**
```
Gross Margin = (Revenue - COGS) / Revenue
```

**Burn Rate:**
```
Monthly Burn = Total Expenses - Revenue
```

**Runway:**
```
Runway (months) = Cash Balance / Monthly Burn
```

- **Pricing:** $0.03 per 1K tokens (input), $0.06 per 1K tokens (output)

- **Pricing:** $0.0023 per image
- **Usage:** 1 image per story
- **Cost per Story:** $0.0023
- **Optimization:** Batch processing, caching
- **Optimized Cost:** $0.002 per story

- **Pricing:** $0.30 per 1K characters
- **Usage:** ~500 characters per story
- **Cost per Story:** $0.15

- **Optimized Cost:** $0.10 per story

- Unoptimized: $22.65
